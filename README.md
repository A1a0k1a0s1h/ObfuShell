## Installation

To install this project:

```bash
  https://github.com/A1a0k1a0s1h/ObfuShell.git
  cd ObfuShell
```
Install the required dependancies:

```bash
  pip install colorama
```
Run the tool:

```bash
  python obfushell.py
```
